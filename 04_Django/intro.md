# 가상환경 설정
```bash
# 파이썬 모듈 중에 'venv' 사용해서
# venv 이름 폴어데 가상환경 만들기
# python -m venv { folder_name }
$ python -m venv venv
```
- 현재 진행할 프로젝트에서 사용할 환경을 구성하기 위해 가상환경 사용
- 다른 환경이나 해당 프로젝트 실행을 위해 필요한 모듈, 패키지 관리를 위해

- 가상 환경 실행
```bash
$ source venv/scripts/activate # (window)
$ source venv/bin/activate # (mac)
```
